# Notes App

Live at https://taskflow-to-do-app.onrender.com/

Deployed with render.com on free tier, so the initial loading may take awhile

Frontend:
- React
- Vite
- Tailwind CSS
- daisy UI
- React Router DOM
- React Redux
- Redux Toolkit
- axios

Backend:
- Node
- ExpressJS
- MongoDB
- Mongoose
- bcrypt
- dotenv
- jsonwebtoken

Features:
- Single Page App
- JWT User Authentication & Authorization (Login / Logout)
- Create tasks
- Edit tasks
- Delete tasks
